=== Description ===
This plugin will help you to integrate Debales AI Assistant Chatbot into your website.

=== Installation === 
1. Upload the plugin files to the directory, or install install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. Go to Settings > Debales Chatbot.
4. Enter your Debales project ID (or project name).
